Bienvenue sur le projet The Musicale.

Projet en 1ère année de BUT Informatique, il nous a été demandé de développer le frontend d'un site en HTML, CSS/BOOTSTRAP, JAVASCRIPT (optionnel).

Nous avons géré des playlists et intégré des balises spotify et pleins d'autres fonctionnalités que je vais vous laisser découvrir.

Il était nécessaire que le site soit responsive.
